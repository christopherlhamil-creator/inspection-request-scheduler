System health & section crosswalk

Everything above assumes a spec section means the same thing in the drawing index, the submittal log, and the inspection record. It doesn't, automatically — the same section shows up as legacy 5-digit code, modern 6-digit code with no separators, or a fully-formatted code with a sub-paragraph locator tacked on, depending on which system logged it. This reconciles all of them to one canonical key, refusing to guess on anything genuinely ambiguous — and then audits what that join actually covers.

      Work package gates

      A package can't open until its submittals are approved, RFIs are clear, and a pre-install meeting was held. And if an engineering change revises the governing drawing after an inspection was recorded, that evidence goes stale — the gate re-locks until it's redone.

      Lessons learned register

      Every recurring inspection-failure pattern from this building gets tallied. Once a pattern clears a threshold, it's promoted from noise to a lesson — and checked against every other building to see whether that trade's scope has started there yet. Not started = the highest-value carry-forward, because the preventive action can still land before the mistake repeats.
